initWithString:

Initializes a HostIPLookupResult object by parsing a string. 

- (id)initWithString:(NSString *)str;
Parameter Descriptions
str
A string returned by the hostip.info API lookup.
Discussion

This initialization method sets up a HostIPLookupResult object by parsing the string representation str, in the format returned by a lookup using the hostip.info API.

latitude

Returns the receiver's latitude. 

- (float)latitude;
Discussion

Latitude is a float value in the range from +90.0 (North Pole) to -90.0 (South Pole), 0.0 being the Equator line.

longitude

Returns the receiver's longitude. 

- (float)longitude;
Discussion

Longitude is a float value in the range +180.0 (East) -180.0 (West), 0.0 being the Greenwich meridian.
