HostIPLookup


Abstract

Performs IP lookup queries using the hostip.info API.

Discussion

The HostIPLookup class provides useful methods to perform lookup queries using the hostip.info API.

Methods

+countryCodeForAddress:
Returns a country code for the desired IP addresses.
+countryCodeForHostname:
Returns a country code for the desired hostname.
+geodataForAddress:
Returns the full query results for the desired address.
+geodataForHostname:
Returns the full query results for the desired hostname.
+lookupAddress:
Returns a HostIPLookupResult object for the desired IP address.
+lookupAddresses:
Returns a HostIPLookupResult objects array for the desired IP addresses.
